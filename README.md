# Employee Handbook Site

A modern, mobile-first employee handbook web app.

## Tech Stack

- Next.js (React)
- SASS/SCSS modules
- Markdown-based content
- Designed for easy overlays and state-based policy display

## Getting Started

Clone the repo, install dependencies, and run locally:

```bash
git clone <repo-url>
cd <repo-folder>
npm install
npm run dev
